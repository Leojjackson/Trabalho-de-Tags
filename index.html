Trabalho da faculdade

Um projeto para desenvolvermos nosso aprendizado de tags passado pelo professor Hebert. Um projeto ou um trabalho muito importante quando se trata de aprender o funcionamento de cada tag e o poder semântico de cada uma dentro de uma página html-5, importante dizer que cada uma possui caracteristica distintas mas com uma importância muito grande dentro de sua página.

Participantes do projeto

Foi determinado pelo professor Herbert que é possivel fazer o trabalho de tags como um grupo, onde todos os participantes do projeto tem determinada responsabilidade seja como desenvolver um paragráfo ou até adicionar e extilizar uma imagem. porém a entrega do trabalho(projeto), deve ser individual. Tentar entender e aprender a aplicar as tags de forma correta em sua página é muito importante pois você começa a aprender o funcionamento de cada uma das tags. por esse motivo deve-se trabalhar na prática o uso de todas elas.

Finalizando o projeto

Uma imagem que foi escolhida como um tema de estudos de programação, onde descreve muitos códigos, onde aprender-los requer tempo e muitos estudos, isso é muito importante para quem deseja aprender, desenvolver, e compartilhar trabalhos e projetos principalmente quando estamos nos tornando desenvolvedores web. estar apto a aprender e gastar tempo em frente ao computador para aprender é muito importante, e quando se trata de apresentar bons trabalhos é muito mais importante.
