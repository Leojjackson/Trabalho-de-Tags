Trabalho da faculdade

Um projeto para desenvolvermos nosso aprendizado de tags passado pelo professor Hebert. Um projeto ou um trabalho muito importante quando se trata de aprender o funcionamento de cada tag e o poder semântico de cada uma dentro de uma página html-5, importante dizer que cada uma possui caracteristica distintas mas com uma importância muito grande dentro de sua página.

Participantes do projeto

Foi determinado pelo professor Herbert que é possivel fazer o trabalho de tags como um grupo, onde todos os participantes do projeto tem determinada responsabilidade seja como desenvolver um paragráfo ou até adicionar e extilizar uma imagem. porém a entrega do trabalho(projeto), deve ser individual.

Finalizando o projeto

Uma imagem que foi escolhida como um tema de estudos, onde descreve o estudo profundo de quem deseja aprender, desenvolver, e compartilhar trabalhos e projetos principalmente quando estamos nos tornando desenvolvedores web. estar apto a aprender e gastar tempo em frente ao computador para aprender é muito importante, e quando se trata de apresentar bons trabalhos é muito mais importante.
